# Leaf-Disease-Detection-using-CNN
Leaf diseases significantly threaten global agriculture by reducing crop yield and quality, leading to economic losses and food insecurity.

Using Python programming and libraries such as TensorFlow and Keras, the project processes images of healthy and diseased leaves to train a CNN model capable of recognizing multiple disease types accurately.
